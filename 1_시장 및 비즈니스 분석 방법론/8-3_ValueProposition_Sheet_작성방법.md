## ✅ **권장 도구 : 페르소나 & JTBD 기반 Value Proposition 통합 문서 구성**

| 항목 | 내용 |
|------|------|
| **페르소나 및 CJM 방식의 고객별 핵심 문제 서술 (Pain, Needs)** | - 페르소나 분석 및 고객 여정지도(CJM)에서 가져온 핵심 Pain, Needs 정리 |
| **JTBD 관점 인터뷰 결과에 따른 고객 상황 기반 목표 (Goal, Job)** | - JTBD 인터뷰 요약 카드에서 추출한 Job Statement & Goal 정리 |
| **고객이 원하는 Outcome** | - JTBD 분석 기반 고객이 달성하려는 목표, 이상적 상태를 **측정 가능하게** 서술 |
| **우리 솔루션의 핵심 제안 (Value Proposition)** | - JTBD + Opportunity 종합 고려하여 솔루션 정의 |
| **기존 대안 (Competitor / Substitute)** | - 경쟁사 분석 결과 기반으로 대안 및 문제점 정리 |
| **우리가 제공하는 차별적 가치 (Differentiation)** | - 솔루션의 고유 강점 및 독창적 가치 제시 |
| **Proof (근거 / 검증 데이터)** | - 정량/정성 데이터 기반으로 Value Proposition을 직접적으로 뒷받침 |

### 👉 **가장 중요한 결과물**
- **솔루션 핵심 가치 제안 (Core Value Proposition)**
- 분석 결과에서 출발하여 **Proof를 통해 설득력 있게 “왜 우리의 솔루션인가?”**를 제시해야 함
- 데이터 기반 시각화 포함 시 효과 극대화
