# **1. Executive Summary**

초기 창업자, 특히 첫 창업을 준비 중인 예비창업자는
**사업계획서 작성, 재무모델 구성, 시장 검증(Market Validation)**에서 가장 큰 어려움을 겪는다.

실제 업로드된 자료 분석 결과:

* **문서 작성 난이도(정부·대출·VC 기준) → 부담 최고**
* **재무 작성에 대한 불안 → 오류·누락 빈번**
* **데이터 기반 시장 검증 부족 → ‘감’에 의존**
* **반복되는 재작업(심사 기준 미달) → 시간 손실**
* **보안 불안 → 온라인 툴 활용 꺼림**

JTBD 인터뷰에서는 특히

> “8시간 이상 걸리는 문서 작업을 1~2시간으로 줄이고 싶다.”
> “재무 작성이 가장 힘들다. 기준이 명확하지 않다.”
> 라는 니즈가 반복 확인되었다.

본 솔루션은 이러한 문제를 해결하기 위해
**문서 자동화 + 재무 자동화 + 데이터 기반 검증**을 핵심 가치로 제공하는
“예비창업자 전용 AI 실행 파트너”이다.

# **2. Target Persona (고객 세그먼트)**

## **2.1 핵심 세그먼트 1 — 과제 수행자 (Segment 2, 8.64만 명)**

**특징**

* 정부지원사업, 대출, 보조금 제출을 앞두고 있음
* ‘하루~3일 내 완료’ 같은 빠른 방식 필요
* 제출 기준을 맞추는 것이 가장 중요한 목표

**주요 Pain**

* 문서 구조를 모르고, 요구 기준도 불명확
* 실수로 떨어질까 불안
* 오탈자·형식 오류 다수
* 밤샘 작업 빈번

**주요 니즈**

* 빠른 자동 생성
* 제출 기준 자동 검수
* 오류 0건 목표

---

## **2.2 핵심 세그먼트 2 — 적극적 학습자 (Segment 3, 4.32만 명)**

**특징**

* 데이터 기반으로 창업 의사 결정을 하려는 타입
* 과거 실패 경험이 있는 경우도 있음
* 리스크 최소화·PMF 검증 중요하게 생각

**주요 Pain**

* 시장 데이터 확보 어려움
* 경쟁사 분석에 시간이 많이 듦
* 검증 없이 제품을 만들기 부담

**주요 니즈**

* PMF 검증 도구
* 시장/경쟁사 자동 조사
* 재무 모델 자동 연동

# **3. Pain & Needs (페르소나 + 고객 여정 기반 분석)**

## **3.1 Pain**

* 정부지원사업 문서가 너무 복잡하고 난도가 높음
* 재무 예측(매출·비용·손익) 작성이 어려움
* 시장 데이터 수집에 시간 소모
* 심사 기준에 대한 이해 부족
* 제출 직전에 오류·오탈자 발견
* 보안 불안(데이터 유출 우려)

## **3.2 Needs**

* 제출 기준에 맞는 **자동 문서 생성**
* 최소 입력으로 완성되는 **재무 자동화**
* 심사 통과 가능성을 높이는 **자동 검수 기능**
* 경쟁사·시장 조사를 자동으로 정리하는 **데이터 기반 분석**
* 안전하게 문서를 보관할 수 있는 **보안 중심 환경**

# **5. Core Value Proposition**

### 💡 **핵심 가치 제안**

**“예비창업자가 겪는 문서·재무·검증의 3대 문제를 AI로 자동화하여,
1~2시간 내 심사 기준에 맞춘 결과물을 제공하는 실행 중심 창업 파트너”**

---

## **5.1 주요 제공 가치**

### **① 사업계획서 자동 생성**

* 제출 기관(정부, 금융, VC) 기준에 맞춘 템플릿 자동 구성
* 최소 입력 기반 요약·구성 자동 완성

### **② 재무 모델 자동화**

* 매출/비용/손익/현금흐름 자동 계산
* 성장률·비용 구조 수정 시 전체 모델 자동 반영

### **③ 데이터 기반 PMF 검증 엔진**

* 시장 규모, 경쟁사, 트렌드 데이터 자동 조사
* 고객 페르소나 기반 문제-솔루션 적합성 평가

### **④ 자동 검수 기능**

* 형식·오탈자·논리 오류 자동 검수
* 제출 전 Risk Alert 제공
